# vlcPlayer

#### 介绍
基于VLC+Qt+Vs2017开发的视频播放器

1.支持本地媒体文件播放

2.支持RTSP流媒体等网络流媒体播放

3.支持声音调整、播放与暂停、播放进度调整

4.支持列表播放，可设置列表循环，随机播放，单一循环
